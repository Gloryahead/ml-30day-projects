# 30-Day Machine Learning Project-Based Curriculum

## Overview
This repository documents a 30-day, project-driven journey from beginner to advanced machine learning practitioner.  
The focus is on **learning by building**, using real-world datasets and professional workflows rather than passive tutorials.

Each day emphasizes:
- reproducibility
- clear reasoning
- clean project structure
- decision-making grounded in data

---

## Tools & Environment
- Python
- VS Code with Jupyter
- pandas, numpy
- matplotlib, seaborn
- scikit-learn
